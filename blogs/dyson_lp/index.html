∫ Linear programming calculator for Dyson sphere program

Kernel:

  • A web-assembly-based linear programming calculator in Dyson sphere program.
  • \(\vec{c}\) is a cost vector associated with each production type.
  • \(\vec{p}\) is a vector of production count.
  • \(\vec{r}\) is a vector of resource count.
  • \(\vec{a}\) is an eligible vector associated with each production type.
  • \(D\) is a matrix of production speed of for resource type x production type.
Minimize: \[ \vec{c}.\vec{p} \] Subject to: \[ \begin{align} 0 &\leq \vec{p} \\ 0 &\leq \vec{r} \\ \vec{p}.\vec{a} &= 0 \\ D.\vec{p} &= \vec{r} \\ \end{align} \]
